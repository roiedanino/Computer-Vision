### Hybrid Images ###

- Start the program (insert the hybridImage.cpp into a Visual Studio (or any c++ environment that open cv can work with) project

- Insert to the console menu the desired photos paths as input

- Move the top trackbar to change the hybrid level (the distance you'll start to see the other low pass image)

- Move the bottom trackbar to change the zoom

- Press 'S' or 's' to swap between the low-pass and high-pass image

- Press 'c' or 'C' to Activate / Disable color
