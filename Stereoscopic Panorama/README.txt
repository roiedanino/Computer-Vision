### Stereoscopic Panoramas ###

- Start the program (insert the StereoscopicPanorama.cpp into a Visual Studio project (or any c++ environment that open cv can work with) 

- Fix the paths in VIDEO_PATHS array to the videos paths in your computer

- Start the program

- If you want you can Insert to the console menu the desired video paths as input

- Choose Frame Frequency, insert 1 for the best results or larger numbers for faster result



* Notice: if you chose 1, its might be pretty slow and you will need to wait some extra minutes for the results.

*If the non free library is not working for you, you can change the feature detection and extraction to BRISK instead
