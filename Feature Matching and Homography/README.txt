### Feature Matching ###

- Start the program (insert the HW3FeatureMatching.cpp into a Visual Studio project (or any c++ environment that open cv can work with) 

- Insert to the console menu the desired photos paths as input

- Choose the detection method, enter 1 for *BRISK or 2 for ORB

- Enter reasonable threshold (Must be a positive integer)


* Notice: if you chose BRISK, its might be pretty slow and you will need to wait some extra seconds for the results.
